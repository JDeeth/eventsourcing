========
Tutorial
========

A tutorial for event sourcing in Python.

This tutorial introduces event-sourced aggregates and event-sourced applications.

It explains just enough of the design and the mechanics of this library
to help users understand basically how things work. It expands and explains
the :ref:`Synopsis <Synopsis>`. It and prepares new users of the library for
reading the :doc:`Modules </topics/modules>` documentation.
